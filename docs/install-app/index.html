Install the scriv-release App

This page registers a private GitHub App in your account or organization with the permissions required by the scriv-release action: contents: write and pull_requests: write. The App is owned by you; scriv-release never sees its private key.

After you click below, GitHub shows a confirmation page on github.com with the App's permissions pre-filled. Click the green "Create GitHub App" button at the bottom to actually create it — the install page only prepares the form. You'll then be redirected to a callback page that shows the App's Client ID and private key once.

You can change the suggested name (scriv-release-bot) on the GitHub confirmation page if it's already taken — GitHub App names are globally unique.

Manual setup (or to inspect the manifest)

If you'd rather create the App by hand, navigate to https://github.com/settings/apps/new (or your org's equivalent) and fill in:

  • Repository permissions: Contents: Read and write
  • Repository permissions: Pull requests: Read and write
  • Subscribe to events: none
